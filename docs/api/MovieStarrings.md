# 电影明星

- 电影明星信息接口

## 查询电影明星详情

- 请求路径：`api/MovieStarrings/:id`
- 请求方法： `get`
- rest参数

  | 参数 |   说明    |  类型  | 可选值 | 默认值 | 是否必填 |
  | :--: | :-------: | :----: | :----: | :----: | :------: |
  | `id` | _id | string |   -    |   -    |    是    |

- 请求响应

  ```json
  {
    "message": "操作成功",
    "data": {}
  }
  ```

## 查询电影明星列表

- 请求路径：`api/MovieStarrings`
- 请求方法： `get`
- query参数

  |    参数     |   说明   |  类型  |    可选值     | 默认值 | 是否必填 |
  | :---------: | :------: | :----: | :-----------: | :----: | :------: |
  |   `page`    |   页码   | number |       -       |   -    |    否    |
  |   `size`    | 每页条数 | number |       -       |   -    |    否    |
  | `sortType`  |   排序   | string | `asc`、`desc` | `desc` |    否    |
  | `sortField` | 排序字段 | string |       -       |   -    |    否    |
  | `startTime` | 创建开始时间 | number |       -       |   -    |    否    |
  |  `endTime`  | 创建结束时间 | number |       -       |   -    |    否    |

- 请求响应

  ```json
  {
    "message": "操作成功",
    "data": {
      "data": [],
      "total": 0
    }
  }
  ```

  | :---------: | :------: | :----: | :-----------: | :----: | :------: |
  |   `id`    |   ID   | string |       -       |   -    |    是    |

- body 参数

  |    参数     |   说明   |  类型  |    可选值     | 默认值 | 是否必填 |

- 请求响应

  ```json
  {
    "message": "操作成功",

  }
  ```
